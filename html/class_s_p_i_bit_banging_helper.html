Detailed Description

Helper-класс для передачи данных по SPI (MSB-first - сначала передаётся старший бит, потом младшие).

Данный класс не знает ничего о конкретном SPI-устройстве. Его единственная задача — корректно передать и принять байт через SPIBitBangingDriver.

uint8_t SPIBitBangingHelper::transferByte ( uint8_t tx_byte)
inline

Передать один байт по SPI и одновременно принять ответный байт.

Передача выполняется в режиме MSB-first. Для каждого бита:

  • выставляется MOSI
  • формируется тактовый импульс SCLK
  • считывается MISO
