Detailed Description

Абстрактный интерфейс SPI драйвера с побитовым управлением.

Класс определяет минимальный набор операций, необходимых для реализации SPI протокола в режиме bit-banging.

Реализации этого интерфейса:

  • управляют GPIO
  • обеспечивают корректные временные интервалы
  • не содержат логики конкретных SPI-устройств
